Data Type & Storage

  1. Integer - Number : 10, 10.50
  2. String - "Hello world" , 'hello world', 'a' , "Z"
  3. Boolean - true or false | 1 or 0
  4. Array - let array = ['apples', "banana" , 10]
  5. Objects - let personData = { name : "Muzaffar" , age : 20, address : "srinagar kashmir", pincode : 190001, interest : ['cricket','football'], maritalStatus : true }

Identifiers: Any named thing is called identifier, by which we can identify variables, function names, or constants, Idenitifiers are case sensitive myVar != myvar
eg, var -> keyword, myVar -> identifierm = -> assignment operator, 10 -> value or literal;


Keywords: Reserved words in any programming lanuguage which we cannot use as identifiers for functionNames constants or variables;

Operators

  1. Arithmetic Operators : + - * /
  2. Assignment Operators : =
  3. String Operators : "Hello" + "Salwinder" (concatenation)
  4. Comparison Operators : ==, ===, !=, !==, > (greater than) , < (less than) , >= (greater than or equal equal to) , <= (less than or equal to)
  5. Ternary Operator : ? : ; -> condition ? true : false; 10 == 10 ? "equal" : "not equal";
