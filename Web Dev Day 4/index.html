ID Attribute

        ID Attribute in html: id="someid"
        can contain numbers, charcters only, & special charcters like _-
        id can't start with number
        id can't contain spaces
        id values are unique only one id can be placed inside 1 web page
        and id's can be accessing using # like #someid

Class Attribute

        class Attribute in html: class="someclass"
        can contain numbers, charcters only, & special charcters like _-
        class can't start with number
        class can't contain spaces
        class values can be multiple and can be used multiple times inside any web page
        and classes can be accessing using . like .someclass
