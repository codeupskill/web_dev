CSS : CASCADING STYLE SHEETS

by using css we can style our web pages. we have 3 diff ways to use css.

  1. Inline CSS :- within the html element using style attribute <h1 style="color:yellow;">
  2. Internal CSS :- can be used withing html page inside head tag iniside style tags <style> </style>
  3. External CSS

by 3 ways we can access an html element:

  1. By tagname : body, h1, h2, p, div
  2. By class name : .someclass
  3. By Id : #someid

Priority of css: 1. Inline , 2. Internal , 3. External
