Events

  • onclick : when user clicks on any element
  • onchange : when an element has been changed
  • onmouseover : when user mouse over an element
  • onkeydown : when user pushed down a key on keyboard
  • onkeyup : when user lifts up a pushed down key
  • onload : when html page has finished it's loading
  • onblur : when user shifts from an html element
